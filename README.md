# Investment Portfolio Construction

## Diploma project on creating an investment strategy to overtake the Moscow Exchange Index (IMOEX)

## User-guide

### Dependencies:

#### Python dependencies:
- `Python>=3.10,<3.12`
- `Poetry>=2.1.2`

#### Installing python packages:
```bash
poetry install
```

#### Run project:

```bash
poetry run python3 main.py
```
